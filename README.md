# Luna - ИИ-администратор для салона красоты

RAG-бот, который отвечает клиентам 24/7, консультирует по услугам и ценам и записывает на приём в Google-календарь. Работает в Telegram и WhatsApp.

## Проблема

Клиент пишет в салон в 3 ночи, чтобы записаться на маникюр. Администратор спит. Без автоматизации шанс быстрого ответа близок к нулю: заявка либо теряется, либо ждёт до утра, а клиент за это время уходит к конкуренту, который ответил.

Держать оператора в чате 24/7 дорого и всё равно не покрывает ночь и выходные.

## Решение

Бот отвечает мгновенно в любое время:

- консультирует по услугам, ценам и мастерам, отвечая по реальной базе салона (не выдумывает)
- понимает голосовые сообщения
- пересчитывает цены в другую валюту по актуальному курсу
- проверяет свободное время в Google-календаре и оформляет запись
- вносит клиента в календарь и присылает уведомление о новой заявке

Результат: заявки не теряются, не нужен оператор чата, клиент получает ответ за секунду.

## Как работает

Клиент пишет боту → AI-агент понимает запрос → при вопросе об услугах достаёт нужный кусок из базы знаний в Qdrant (RAG) и отвечает по факту → при записи проверяет Google-календарь на свободное время → создаёт событие и уведомляет салон о новой заявке.

Пример из демо: клиент просит записать его на 3 ночи, бот отказывает (салон закрыт) и предлагает ближайшее свободное время в рабочие часы.

## Стек

- **n8n** (self-hosted, Hetzner) - оркестрация, AI Agent
- **Qdrant** - векторная база знаний, RAG (коллекция услуг, цен, мастеров)
- **OpenRouter / GPT-4o** - модель, генерирующая ответы
- **Google Calendar (OAuth2)** - проверка занятости и запись клиентов
- **GreenAPI** - канал WhatsApp
- **Telegram Bot API** - канал Telegram

## Ключевые технические моменты

- **RAG на Qdrant**: перед ответом об услугах агент подтягивает релевантные данные из базы, поэтому цены и мастера всегда реальные, а не сгенерированные
- **Agent + tools**: конвертация валюты по актуальному курсу как отдельный инструмент агента
- **Двухканальность**: один воркфлоу обслуживает Telegram и WhatsApp
- **OAuth2** для безопасного доступа к Google-календарю
